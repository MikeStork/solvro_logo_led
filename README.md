# Solvro Logo PCB

**Solvro Logo PCB** to dekoracyjna płytka PCB w kształcie logo **Strategicznego Koła Naukowego Solvro**.  
Projekt powstał jako element promocyjny oraz demonstracja możliwości projektowania PCB.
![wizualizacja_końcowa](https://github.com/MikeStork/solvro_logo_led/blob/master/wizualizacja_ko%C5%84cowa.PNG)

---

## Cel projektu

Celem projektu było stworzenie **estetycznej płytki PCB**, która przedstawia logo Solvro i może być używana jako:

- gadżet promocyjny
- demonstracja projektowania PCB
- ozdoba stanowiska pracy
- przykład dla nowych członków sekcji hardware

---

## Projekt

Płytka została zaprojektowana tak, aby **warstwy PCB tworzyły grafikę logo Solvro**.  
Elementy takie jak:

- led
- soldermask
- miedź
- silkscreen

zostały wykorzystane do uzyskania odpowiedniego wyglądu logo.

Projekt pokazuje, że PCB może być używane nie tylko do elektroniki, ale również jako **nośnik grafiki i designu**.

---

## Technologie

Projekt został wykonany przy użyciu:

- **KiCad**
- standardowej technologii **PCB FR4**
